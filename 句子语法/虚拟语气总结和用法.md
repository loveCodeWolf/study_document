# 虚拟语气 · 超详细完整版速查表

---

# 一、If 条件句虚拟（最核心）
## 1. 与**现在**事实相反
- 从句：If + 主语 + **did / were**
- 主句：主语 + **would / could / should / might + do**
- 含义：**现在其实不是这样**
- 翻译：如果现在……，就会……
- 例句：
  If I **were** you, I **would accept** it.
  如果我是你，我就会接受。

---

## 2. 与**过去**事实相反
- 从句：If + 主语 + **had done**
- 主句：主语 + **would / could / might + have done**
- 含义：**过去没做，现在后悔/假设**
- 翻译：要是当时……，本来就会……
- 例句：
  If I **had studied** harder, I **would have passed**.
  要是我当时更努力，我本来会通过的。

---

## 3. 与**将来**事实相反（可能性极小）
三种从句都可以：
1. If + **did**
2. If + **should do**（万一）
3. If + **were to do**
- 主句：would / could / might + do
- 翻译：万一将来……，就会……
- 例句：
  If it **should rain** tomorrow, we **would cancel** the trip.
  万一明天下雨，我们就会取消行程。

---

# 二、wish / if only（后悔、但愿）
## 1. 对**现在**愿望
- wish + **did / were**
- 翻译：我真希望现在……
- I wish I **were** taller.
  我真希望我现在更高。

## 2. 对**过去**后悔（考试最爱）
- wish + **had done**
- 翻译：我真希望当时……
- I wish I **had not said** that.
  我真希望我当时没说那句话。

## 3. 对**将来**希望
- wish + **would / could do**
- I wish you **would come** earlier.
  我希望你能早点来。

---

# 三、表示 **建议、命令、要求** 的虚拟
**一律：(should) + 动词原形**

### 必须背的动词（一个都不能漏）
suggest / advise / propose / recommend
demand / require / request
order / command
insist

### 结构
- 动词 + (that) + 主语 + **(should) do**
- 翻译：建议/要求/命令……应该做……

例句：
The teacher suggested that we **(should) review** the notes.
老师建议我们复习笔记。

### 形容词虚拟（同结构）
It is important / necessary / natural / strange
+ that + 主语 + (should) do

It is necessary that he **(should) go** at once.
他有必要立刻去。

---

# 四、would rather 宁愿
## 1. 宁愿某人**现在/将来**做
- would rather + 主语 + **did**
- I would rather you **came** tomorrow.
  我宁愿你明天来。

## 2. 宁愿某人**当时**做（过去）
- would rather + 主语 + **had done**
- I would rather you **had told** me the truth.
  我宁愿你当时告诉了我真相。

---

# 五、It is (high) time 该做……了
- It is (high) time + 主语 + **did**
- 翻译：是时候……了（早该做了）
- It is high time we **went** home.
  我们该回家了。

---

# 六、as if / as though 好像
## 1. 与**现在**相反
- as if + **did / were**
- He talks as if he **knew** everything.
  他说话好像什么都知道。

## 2. 与**过去**相反
- as if + **had done**
- He looked as if he **had seen** a ghost.
  他看起来好像见了鬼一样。

---

# 七、含蓄虚拟（没有 if 的虚拟）
## 1. without / but for 要不是
- Without your help, I **would have failed**.
  要不是你的帮助，我本来会失败。

## 2. otherwise / or 否则
- I was busy. Otherwise I **would have helped** you.
  我当时很忙，否则我就帮你了。

---

# 八、虚拟语气快速判断口诀（必背）
- 现在虚拟：**过去式**
- 过去虚拟：**had done**
- 建议命令：**(should) do**
- wish 后悔：**现在用过去，过去用 had done**
- It’s time：**一律过去式**
- would rather：**现在过去式，过去 had done**
